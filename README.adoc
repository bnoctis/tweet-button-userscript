= 鳴叫！

将您 Twitter 網頁版的“推文”按鈕改爲“鳴叫！”。

== 使用

這是一個 user script。您需要安裝一個 user script manager 如：

- TamperMonkey
- GreasyMonkey
- ViolentMonkey

打開 https://raw.githubusercontent.com/bnoctis/tweet-button-userscript/main/%E9%B3%B4%E5%8F%AB%EF%BC%81.user.js
，並按照您的 manager 提示安裝。
